# nn_tools

本 package 用于探索：能否让预训练网络识别数据中的模式，并在未来作为一种
模式提取器加入到 `src/sr_agent/tools/` 中。

可能的任务包括两类。一方面，可以尝试从数据中识别公式里可能存在的符号结构；
另一方面，也可以尝试识别公式的函数性质，例如某个变量是否为冗余变量、因变量
关于某个自变量是否具有周期性等。具体让模型预测哪些信息最有价值，还需要继续探索。

可能的技术路径也暂不固定。例如，可以使用预训练模型作为 Encoder，再接一个用于
特征提取或属性预测的 Decoder；也可以直接端到端训练一个模型，从输入数据预测目标
Label。

最终目标是：输入一组数据，输出一些关于这组数据的有价值事实，从而帮助 Agent 更好地
进行符号回归。
